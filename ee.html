A simple enough concept but apparently against the design philosophy of most 2000's blog pages.

Note to reader and to self: Google chrome has been apparently following an anti-autoplay policy since 2017 (Google Chrome Autoplay Policy Changes Page). The music is autoplaying just fine in other browsers like MS Edge and IE Explorer. If you are using Google Chrome and actually clicked somewhere on the page, the current music autoplay "magic" that you have just witnessed is actually a script that starts the music as soon as the user clicks anywhere on the page.

This autoplay "feature" allows us replicate the so very popular (and loud) 00's websites.
